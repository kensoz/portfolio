## スタック

- ⚡️ Nuxt.js 3
- ⚙️ TypeScript
- 🎨 TailwindCSS 3
- 📑 Eslint + Prettier
- 📦 Yarn

## 使用

##### インストール

```bash
yarn install
```

##### 開発

[http://localhost:3000](http://localhost:3000)で起動

```bash
yarn dev
```

##### server build

```bash
yarn build
```

##### pubic build

```bash
yarn generate
```

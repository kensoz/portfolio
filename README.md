個人ポートフォリオサイトです。  
🏠 **[Site](https://kensoz.github.io/portfolio/)**

## スタック

- ⚡️ Vue.js

- ⚙️ TypeScript

- 🎨 TailwindCSS

  

## 使用

##### インストール

```bash
yarn install
```

##### 開発

[http://localhost:3000/portfolio](http://localhost:3000/portfolio/)で起動

```bash
yarn dev
```

##### ビルド

```bash
yarn build
```

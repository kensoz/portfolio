ポートフォリオサイトです。  

## スタック

- ⚡️ Vue.js

- ⚙️ TypeScript

- 🎨 TailwindCSS

  

## 使用

##### インストール

```bash
yarn install
```

##### 開発

```bash
yarn dev
```

##### ビルド

```bash
yarn build
```
